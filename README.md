# Nova Curriculos

## Execute project

    git clone git@bitbucket.org:luizpicolo_/nova-curriculos.git
    bundle install
    rake db:create
    rake db:migrate
    rails s

    open http://localhost:3000

## Dependencies

- Ruby 2.1.0+
- Mysql 5.5+ or Postgre 9.3+ DataBase
- rvm 1.25.x+ (optional)
